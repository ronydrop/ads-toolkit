# /copy-meta — Criar Copy para Meta Ads

Carregar skills: ad-copy-principles, account-conventions.

## Workflow
1. Perguntar: produto? público alvo? ângulo preferido?
2. Puxar criativos que mais performaram (analyze_meta_ad_performance, 14 dias)
3. Identificar ângulo vencedor atual
4. Gerar 3 variações de copy (A/B/C) com:
   - Headline: 1 linha, punch imediato
   - Corpo: 2-4 linhas para mobile
   - CTA: explícito e ativo
5. Indicar qual ângulo testar primeiro baseado no histórico

Ângulos disponíveis: social proof, pattern interrupt, humor, polêmico, urgência
