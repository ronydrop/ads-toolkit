# /budget — Otimização de Orçamento

Carregar skills: budget-optimize, performance-analysis, account-conventions.

## Workflow
1. Puxar performance 7 dias de todas as campanhas ativas (Meta + Google)
2. Calcular CPL/CPA de cada campanha
3. Identificar: campanhas "limited by budget" com bom CPL
4. Identificar: campanhas com CPL ruim há 3+ dias
5. Calcular redistribuição: tirar de ruins, colocar nas boas
6. Apresentar tabela antes/depois para aprovação
7. NUNCA executar mudança sem confirmação do Rony
